# The Journey of Learning: Gagné's 9 Events

An interactive 3D educational game built in Unity, designed to visualize and teach **Robert Gagné's 9 Events of Instruction**. Instead of reading dry theory, players step into the shoes of a student navigating a low-poly archipelago, where each island represents a core stage of the learning process.

## 📖 About the Project
This project was developed as a university assignment to bridge the gap between Game Design and Instructional Design. The goal is to prove that gamification and interactive environments can make abstract educational theories engaging and easy to understand. 

The player must interact with 9 different NPCs across 9 levels. Each NPC explains a specific Gagné Event, triggering a unique 3D visual effect (particles, holograms, dynamic lighting) that metaphorically represents the cognitive process of that event.

## ✨ Key Features
* **9 Unique Learning Stages:** Each level is uniquely tied to one of Gagné's events (e.g., Gain Attention, Elicit Performance, Provide Feedback).
* **Dynamic Interactions:** Custom C# dialogue system using math-based distance calculations (bypassing standard collider bugs) for precise NPC interaction.
* **Metaphorical VFX:** * *Spotlights* for presenting content.
  * *Floating Holograms* for providing feedback.
  * *Glowing Trails* for learning guidance.
  * *Particle Explosions* for retention and transfer.
* **Custom UI/UX:** Clean, delayed-text UI panels using TextMeshPro, complete with an End-Game screen.

## 🛠️ Built With
* **Game Engine:** Unity 3D
* **Scripting:** C#
* **UI:** Unity Canvas & TextMeshPro
* **Assets:** Modular Low Poly Environment

## 🎮 How to Play
* **W, A, S, D:** Move around the world.
* **Mouse:** Look around.
* **E:** Interact with NPCs to start the dialogue and trigger the learning events.
* Read the dialogues, observe the visual metaphors, and watch the barriers drop to proceed to the next island!
